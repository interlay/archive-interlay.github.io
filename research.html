XCLAIM (2019)

IEEE Symposium on Security and Privacy (S&P'19)
We propose XCLAIM, the first framework for financially trustless cross-chain transfers using cryptocurrency-backed assets (CbAs). XCLAIM allows to create assets which are 1:1 backed by existing cryptocurrencies, without requiring trust in a central operator, e.g. Bitcoin-backed tokens on Ethereum.

Read the paper

The Scalability of Trustless Trust (2018)

Trusted Smart Contracts Workshop, Financial Cryptography and Data Security 2018 (WTSC FC'18)
Permission-less blockchains can realise trustless trust, albeit at the cost of limiting the complexity of computation tasks.We propose a trust model for smart contracts, described as agents in an open multi-agent system.

Read the paper

Merged Mining: Curse or Cure? (2017)

Data Privacy Management, Cryptocurrencies and Blockchain Technology (CBT'17)
We show that merged mining, the concept of mining more than one cryptocurrency in parallel by reusing Proof-of-Work, leads to centralization across blockchains. We introduce a simple but effective scheme to attribute blocks to miners and present the first technique to link miner payout addresses cross-chain.

Pre-Prints

SoK: Communication Across Distributed Ledgers (2020)

Cryptology ePrint Archive
We provide the first systematic exposition of protocols for cross-chain communication and show cross-chain communication is impossible without a trusted third party, contrary to common beliefs in the blockchain community. We develop a framework for evaluating existing and designing new cross-chain protocols, based on use case, trust model and security assumptions of interlinked blockchains.

Read the paper

Pay-to-Win: Incentive Attacks on Proof-of-Work Cryptocurrencies (2019)

Cryptology ePrint Archive
We systematically analyze and categorize bribing attacks and similar techniques (incentive attacks) in proof-of-work cryptocurrencies. We then propose the first trustless cross-chain bribing attack, in which miners face no risk and hence will always participate, if economically rational. Our attack inter alia allows to coordinate double spending attacks across different blockchains and can be crowdfunded, minimizing costs per colluding attacker.
