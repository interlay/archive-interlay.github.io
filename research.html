Our Open Source Research Impact

A list of selected academic publications previously authored by Interlay team members, pioneering research in blockchain interoperability, security, and game theory.

XCLAIM (2018)

IEEE Symposium on Security and Privacy (S&P'19)
We propose XCLAIM, the first framework for financially trustless cross-chain transfers using cryptocurrency-backed assets (CbAs). XCLAIM allows to create assets which are 1:1 backed by existing cryptocurrencies, without requiring trust in a central operator, e.g. Bitcoin-backed tokens on Ethereum.

SoK: Communication Across Distributed Ledgers (2020)

Cryptology ePrint Archive
We provide the first systematic exposition of protocols for cross-chain communication and show cross-chain communication is impossible without a trusted third party, contrary to common beliefs in the blockchain community. We develop a framework for evaluating existing and designing new cross-chain protocols, based on use case, trust model and security assumptions of interlinked blockchains.

arXiv.org

The Scalability of Trustless Trust (2018)

Cryptology ePrint Archive
Permission-less blockchains can realise trustless trust, albeit at the cost of limiting the complexity of computation tasks.We propose a trust model for smart contracts, described as agents in an open multi-agent system.
